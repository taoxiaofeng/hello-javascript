与医院(his)对接的内容(以下内容仅供简单参考，具体细节请查看逸曜审方客户端B/S架构SDK整合方案文档)

步骤一： 在html的head标签中依次引入json2.js,jquery.min-1.10.1.js,hzyy_sdk_20200911.js这三个文件
(注：同级目录下引入消息提示图标资源：info-fill.png) 
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>某某医院信息系统</title>
                <script src="./json2.js"></script>
                <script src="./jquery.min-1.10.1.js"></script>
                <script src="./hzyy_sdk_20200911.js"></script>
            </head>

            <body>
                <script>
                    // 调用 hzyy_sdk_20200911.js 中的accessToTask()方法把 deptId, userId, userName,
                    // 轮序获取消息的url，等参数传入此方法；
                    // 示例：
                    accessToTask(deptId, userId, userName, url);
                </script>
            </body>
